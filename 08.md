### switch

```mermaid
flowchart LR
    A[Instruções] --> C{caso 1}
    C -->|Verdadeiro| D[Instruções]
    C -->|Falso| F{caso 2}
    D --> DB[break]
    DB --> G[Instruções]
    F -->|Verdadeiro| H[Instruções]
    H --> HB[break]
    HB --> G[Instruções]
    F -->|Falso| F2{default}
    F2 --> f3[Instruções]
    f3 --> G[Instruções]
```

``` java
instruções;
switch(VARIAVEL){
  case 1:
      instruções;
      break;
  case 2:
      instruções;
      break;
  default
      instruções;
}
instruções;
```
